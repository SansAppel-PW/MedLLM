# 医疗安全策略（Draft）

## 1. 高风险范围
- 用药剂量、禁忌症、相互作用、急危重症分诊建议。
- 涉及孕妇、儿童、老人、肝肾功能异常等脆弱人群的建议。

## 2. 输出控制策略
- 检测结果为 `High`：默认拒答具体医疗处置建议，改为提示线下就医。
- 检测结果为 `Medium`：输出低风险信息 + 明确不确定性提示 + 建议复核来源。
- 检测结果为 `Low`：允许输出，同时附带证据来源摘要。

## 3. 禁止行为
- 生成具体处方或替代临床医生诊断。
- 对高风险症状给出延误就医建议。
- 编造不存在的药物、指南、临床试验结论。

## 4. 审计与可追溯
- 每次回答记录：输入、模型版本、风险分级、证据片段、拦截动作。
- 高风险命中样本进入人工复核池，用于后续偏好对齐训练。

## 5. 对外声明模板
- 本系统仅用于科研与辅助信息检索，不构成医疗诊断或治疗建议。
- 若出现急性症状或用药相关问题，请及时咨询持证医生。
